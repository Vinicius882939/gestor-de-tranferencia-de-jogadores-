# ⚽ Gestão de Transferências de Jogadores

Sistema focado no mercado de transferências no futebol, permitindo gerir jogadores, clubes, empresários e transferências.

---

## 📌 Descrição

Este projeto tem como objetivo desenvolver um sistema de gestão que simula o mercado de transferências no futebol, permitindo o registo, consulta e manipulação de dados através de operações CRUD.

---

## 🎯 Objetivos

- Gerir jogadores  
- Gerir clubes  
- Gerir empresários  
- Registar transferências  
- Consultar histórico de transferências  
- Aplicar operações CRUD de forma eficiente  

---

## 🧩 Entidades

- 👤 Jogador  
- 🔄 Transferência  
- 🏟️ Clube  
- 🤝 Empresário  

---

## 🔗 Relações entre Entidades

- Um jogador pode ter várias transferências ao longo da sua carreira  
- Cada transferência está associada a um único jogador  
- Um jogador está associado a um empresário  
- Um empresário pode representar vários jogadores  
- Um clube pode comprar e vender vários jogadores  

---

## 🔄 Operações CRUD

### 👤 Jogador
- Criar jogador  
- Listar jogadores  
- Atualizar jogador  
- Remover jogador  

### 🔄 Transferência
- Criar transferência  
- Listar transferências  
- Atualizar transferência  
- Remover transferência  

### 🏟️ Clube
- Criar clube  
- Listar clubes  
- Atualizar clube  
- Remover clube  

### 🤝 Empresário
- Criar empresário  
- Listar empresários  
- Atualizar empresário  
- Remover empresário  

---

## 💡 Funcionalidades

- Registo de transferências (origem → destino)  
- Histórico de transferências por jogador  
- Cálculo de gastos por clube  
- Identificação da transferência mais cara  
  
---

## 📚 Conceitos Aplicados

- Estruturas de dados  
- Programação orientada a objetos  
- CRUD (Create, Read, Update, Delete)  
- Relações entre entidades
